# UI Pro Guidelines
Guidelines for professional-grade user interfaces with Tailwind CSS and React 19.

- **Floating Labels**: Always use floating labels for input fields to create elegant form interfaces.
  - Form inputs are surrounded by relative containers.
  - Placeholder is set to " " and label floats when focused or has content.
- **Micro-interactions**: Hover effects, scale animations, and subtle gradient shifts on CTA elements.
- **Transitions**: Smooth page routing transitions and responsive, staggered list entries using Framer Motion.
- **Aesthetic**: Modern, spacious layout with rounded-3xl/4xl containers, fine borders, and balanced typography.
