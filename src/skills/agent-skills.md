# Agent Skills
This skill file details core guidelines for agents when building applications.

- **Routing**: Client-side routing with `react-router-dom`. Use standard subpaths and route transitions.
- **Components**: Functional, typed React components. Use Lucide React icons.
- **Animations**: Fluid Framer Motion transitions (staggers, slide-ins, scales).
- **Form Handling**: Use modern React 19 paradigms where applicable, structured client-side validations, and responsive interfaces.
